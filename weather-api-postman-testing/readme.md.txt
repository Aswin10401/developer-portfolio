# Weather API Testing with Postman

## Overview
This repository contains Postman collections and environment files for testing the OpenWeatherMap API.

## Getting Started

1. Clone or download this repo.
2. Open Postman.
3. Import the `WeatherAPI.postman_collection.json` file into Postman (File > Import).
4. Import the `WeatherAPI.postman_environment.json` file as an environment.
5. Update the `API_KEY` variable in the environment with your own OpenWeatherMap API key.
6. Select the imported environment in Postman.
7. Run the requests in the collection. Each request contains automated tests that validate the response.

## APIs Tested

- Current Weather Data
- 5-Day Weather Forecast

## Tools Used

- [Postman](https://www.postman.com/)
- [OpenWeatherMap API](https://openweathermap.org/api)
